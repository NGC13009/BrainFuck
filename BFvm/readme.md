# 运行在WindowsNT上的BF解释器

## 编译

    PS > gcc .\BF.c -o BF -fexec-charset=gbk
    PS > gcc .\BF.c -o BF_DBG -D DEBUG -fexec-charset=gbk   //DEBUG模式

--------------

## 使用说明

    PS > ./BF.exe [here insert BF code]
